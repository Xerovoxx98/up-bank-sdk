# Getting Started

This section will help you get up and running with the up-bank-sdk.

## What's Covered

- [Installation](installation.md) — Install the package and dependencies

## Prerequisites

You'll need:

1. A Up Bank account
2. A personal access token from the Up app
3. Python 3.12 or higher
